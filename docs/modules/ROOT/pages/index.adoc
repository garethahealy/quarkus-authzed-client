== Installation

If you want to use this extension, you need to add the `io.quarkiverse.quarkus-authzed-client:quarkus-authzed-client` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.quarkus-authzed-client</groupId>
    <artifactId>quarkus-authzed-client</artifactId>
</dependency>
----
